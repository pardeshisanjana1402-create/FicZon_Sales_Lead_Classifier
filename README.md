## FicZon_Sales_Lead_Classifier
A machine learning project to predict sales lead conversion using classification models and data analysis techniques.

### Database Connection (MySQL)

The dataset was connected using a MySQL database and loaded into Python for further analysis and model building.
### Steps
* Established connection to MySQL database
* Queried required data
* Loaded data into a Pandas DataFrame
* Closed the connection after fetching data

---
### Project Goal

* To build a machine learning model that predicts whether a sales lead will convert into a customer, helping businesses focus on high-potential leads.

---
### Data Preprocessing

* Handled missing values
* Converted date columns into proper format
* Encoded categorical variables
* Removed duplicates
* Prepared clean dataset

---
### Exploratory Data Analysis (EDA)

* Visualized lead conversion trends
* Identified key factors affecting conversion
